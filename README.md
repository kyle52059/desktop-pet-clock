# 🐱 桌面宠物时钟 Desktop Pet Clock

一个可爱的桌面宠物时钟应用！

## ✨ 功能

- 💕 **系统托盘图标** - 最小化到右下角，不占用任务栏
- ⏰ **桌面透明时钟** - 显示时间和日期，可拖动
- 🐱 **可爱小猫咪** - 在桌面上走来走去，会眨眼、摇尾巴
- 🖱️ **可拖动** - 时钟和宠物都可以拖到任意位置

## 📦 安装依赖

```bash
pip install PyQt5 pyinstaller
```

## 🚀 运行

```bash
python pet_clock.py
```

## 📦 打包成 EXE

```bash
pyinstaller --onefile --windowed --icon=icon.ico --name="桌面宠物时钟" pet_clock.py
```

打包后的 exe 文件在 `dist` 文件夹中。

## 🎮 使用方法

1. 运行程序后，右下角会出现一个粉色爱心图标
2. 桌面上会显示时钟和一只粉色小猫咪
3. 右键点击托盘图标可以：
   - 显示/隐藏时钟
   - 显示/隐藏宠物
   - 退出程序
4. 可以拖动时钟和宠物到任意位置

## 💕 特色

- 粉色可爱风格
- 小猫咪会随机走动、眨眼、摇尾巴
- 透明背景，不遮挡桌面
- 轻量级，占用资源少
